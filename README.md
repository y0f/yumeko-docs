# Yumeko Docs

Player guide for the Yumeko Discord bot, built with [Astro Starlight](https://starlight.astro.build/).

Live at https://y0f.github.io/yumeko-docs/

## Develop

```
npm install
npm run dev
```

Pages live in `src/content/docs/` as plain Markdown. Sidebar config is in
`astro.config.mjs`. Pushing to `main` deploys to GitHub Pages automatically.
