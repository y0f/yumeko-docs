Yumeko also carries a toolbox of fun and utility commands.

## Fun

| Command | What it does |
| --- | --- |
| `/8ball <question>` | Ask the magic 8-ball |
| `/fight <opponent>` | Duel another player (or the bot). No money involved, only honor |
| `/urban <term>` | Urban Dictionary lookup |
| `/redgpt <question>` | Ask the bot's AI anything |
| `/poll` | Create a poll |

## Utility

| Command | What it does |
| --- | --- |
| `/avatar` | Show anyone's avatar full size |
| `/userinfo` / `/serverinfo` | Account and server details |
| `/translate <text>` | Translate text (also as right-click on any message) |
| `/define <word>` | Dictionary definition |
| `/timestamp` | Discord timestamp generator |
| `/encode <text> <format>` | Base64, URL, hex, ROT13, binary, reverse |
| `/hash <text>` | Hash a string |
| `/ocr <image>` | Read text out of an image |
| `/screenshot <url>` | Screenshot a webpage |
| `/addemoji` | Add an emoji to the server (also as right-click Steal Emoji) |
| `/ping` | Is the bot alive |

## For moderators

Server staff get moderation tools too: message purging, timeouts, slowmode,
channel lock and unlock, plus `/snipe` and `/editsnipe` to reveal recently
deleted or edited messages. These need the right Discord permissions, regular
players will not see most of them.
