Everything you do with Yumeko earns XP.

## Earning XP

| Action | XP |
| --- | --- |
| Using a command | +5 |
| Winning a game | +25 |

There is a short cooldown (about a minute) between XP grants, so spamming
commands does not farm levels.

## Levels

Each level needs more XP than the last, starting around 100 XP for level 1 and
growing steadily. Servers can attach rewards to levels: cash drops, special
roles, or both. Ask your server's staff what is configured.

## Commands

| Command | What it does |
| --- | --- |
| `/rank` | Your level, XP progress and rank (works on others too) |
| `/levels` | The server's level leaderboard |
| `/stats` | Includes a leveling page next to your money and gaming stats |
